# Screenshots

This folder should contain the following screenshots:

1. **login_screen.png** - Screenshot of the login/registration screen
2. **chat_screen.png** - Screenshot of the chat interface with messages
3. **firestore_console.png** - Screenshot of Firebase Firestore console showing the chat collection

## How to Take Screenshots

### Login Screen
1. Run the app
2. Take a screenshot of the login screen showing email/password fields and buttons

### Chat Screen
1. Login to the app
2. Send a few test messages
3. Take a screenshot showing the chat interface with messages

### Firestore Console
1. Go to Firebase Console (https://console.firebase.google.com/)
2. Navigate to Firestore Database
3. Open the "chats" collection
4. Take a screenshot showing the stored messages

## Screenshot Requirements
- Use PNG format
- Ensure text is readable
- Show relevant UI elements clearly
- Remove any sensitive information before committing
